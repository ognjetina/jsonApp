This app was made to hould our json data for app test or something similar on our local server. Take in mind that jsons are in memory so if the server shuts down jsons will be lost!

Usage:

  • You can take a look at all jsons at link: all jsons

  • You can add your json by sending POST or PUT method at /add_json url with your json and server will tell you your json id.

  • You can edit your json by sending POST or PUT method at /edit_json with your new json data.
    Just add "jsonId" field to your new json with id of json you are editing (no worries jsonId field will not be added).

  • You can delete your json by sending GET method to /delete_json with parameter jsonId.

  • You can get your json by sending GET method to /get_json with parameter jsonId.
